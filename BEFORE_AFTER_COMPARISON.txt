USER JOURNEY EXAMPLE:
================================================================================

Scenario: Finding a Bangla book under ৳220 in Story Book category

Step 1: Click "Story Book" from navbar
        → Lands on /category/story-book/

Step 2: View the filters sidebar
        → Sees: Price Range, Language, Author (Kalpona), Publisher

Step 3: Select Language: "Bangla"
        → Page auto-refreshes, shows only Bangla books

Step 4: Enter Min Price: 100, Max Price: 220
        → Click "Apply Filters"

Step 5: View filtered results
        → Sees only Bangla books priced between ৳100-220
        → In this case: No results (Kalpo Galpo is ৳250)

Step 6: Adjust filters
        → Change Max Price to 300
        → Click "Apply Filters"
        → Now sees "Kalpo Galpo"

Step 7: Sort by price
        → Select "Price: Low to High"
        → Books sorted, filter stays applied

Step 8: Clear filters
        → Click "Clear Filters"
        → Returns to all books in Story Book category

TECHNICAL IMPLEMENTATION HIGHLIGHTS:
================================================================================

1. Django View (category_books):
   ```python
   # Get books from this category
   books = Book.objects.filter(is_active=True, category=category)
   
   # Apply filters
   if language: books = books.filter(language=language)
   if author: books = books.filter(author=author)
   if publisher: books = books.filter(publisher=publisher)
   if min_price: books = books.filter(price__gte=min_price)
   if max_price: books = books.filter(price__lte=max_price)
   
   # Get unique values for dropdowns (category-specific)
   authors = books.values_list('author', flat=True).distinct()
   publishers = books.values_list('publisher', flat=True).distinct()
   ```
